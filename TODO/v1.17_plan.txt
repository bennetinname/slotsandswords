 0) MECHANIK-ÄNDERUNGEN (Fundament – zuerst bauen)
====================================================================

[M1] SCHILD/BLOCK VERFÄLLT (Slay-the-Spire-Regel)
  - Aktuell: Block bleibt über Runden erhalten (game._start_combat/Logik:
    "Block bleibt erhalten!"). Neu: Block wird zu Beginn der NÄCHSTEN
    Spielerrunde auf 0 gesetzt (Spieler blockt -> Gegner schlägt -> wenn
    Spieler wieder dran ist, ist der alte Block weg).
  - Umsetzung: in _start_new_turn() VOR dem Nachziehen player.block = 0.
    (Wächter-Relikt/Ritter-Buff +Block kommt DANACH, gilt also frisch.)
  - WICHTIG: Block-Payoff-Karten (Schildschlag/Bodyslam) und "Block
    behalten"-Effekte werden dadurch erst interessant -> siehe Archetyp D.
  - Save-kompat: rein Verhalten, keine neuen Pflichtfelder.

[M2] GLÜCK AUFWERTEN (Glücksdreher müssen mehr bringen)
  - Aktuell: pro Glücksdreh 30% je Walze auf {HEART,MONEY,STAR,CROWN}.
    Zu schwach, kennt neue Symbole nicht, kein Feedback.
  - Neu:
    * Glücksdreh unterdrückt NEGATIV-Symbole: TRAP/PIT/CURSE erscheinen in
      einem Glücksdreh gar nicht (bzw. werden ersetzt).
    * Stärkerer Bias: ~55% je Walze auf ein "gutes" Symbol; gute Liste
      erweitern um WILD/BELL/GEM (+ HEART/MONEY/STAR/CROWN/CLOVER).
    * Sichtbares Feedback: "🍀 GLÜCKSDREH" Banner + Klee-Glow an den Walzen,
      eigener Sound.
    * Optional: pro übriger Glücksrunde kleiner Flat-Bonus auf Slot-Gold.
  - Damit wird Glück eine echte Build-Ressource (Croupière/Hochstapler).

[M3] MULTIPLIKATOR-STAT (für Archetyp G)
  - Neuer Kampf-Stat player.mult (Default 1, Reset pro Kampf wie Stärke).
  - Aufbau-Karten erhöhen mult; "Finale"-Karten: Schaden = Basis * mult,
    danach mult zurück auf 1. Im Status-Chip anzeigen (✖️).

[M4] STATUS-DETONATION (Payoff-Muster für Gift/Brennen)
  - Karten, die einen Status "entfesseln": Schaden = Status * Faktor,
    Status wird verbraucht. Gibt jedem Status-Build einen Höhepunkt.

[M5] SLOT-MANIPULATION (Kartentyp, Archetyp F – euer Alleinstellungsmerkmal)
  - Karten, die VOR dem Dreh wirken: Walze sperren (Symbol fixieren),
    Walze neu drehen, Wunschsymbol einschmuggeln, Dreh zählt doppelt.
  - Braucht etwas UI (Walze/Symbol auswählen). Klein anfangen (1-2 Karten),
    Spielgefühl prüfen, dann ausbauen.
